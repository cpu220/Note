二叉树有几种遍历方式？

常用的方式： 深度优先遍历 和 广度优先遍历 两种， 
其中深度优先遍历氛围前序、中序、后续三种

# 什么是二叉树？

1. 最顶层的节点叫 **根节点**
2. 没有子节点的节点，叫**叶子节点**


```

    A   <-- 根节点
   / \
  B   C <-- 第二层节点
 / \
D   E <-- 第三层节点（叶子节点）
```

# 深度优先遍历

深度优先遍历总共有三种
也就是一条路走到头，遇到叶子节点后，再回头走其他路径。核心是： **先深后广**
三种便利的区别就只是，访问**根节点**的时机不同，记住【根】的位置，就能分清三种便利

## 1. 前序遍历 （根->左->右）

所谓的前序，就是根在前，先处理根，然后再处理左右

```
A -> B -> D -> E -> C
```

```js

const inorderTraversal = function (root) {
 
    const arr = []
    const deep = (root) => {
        if (!root) {
            return
        }
        
        arr.push(root.val)
        deep(root.left) 
        deep(root.right)

    }
    deep(root)

    return arr


};

```

## 2.中序遍历 (左->根->右)

所谓中序，就是根在中间，先处理左，再处理根，再处理右

```
D -> B -> E -> A -> C
```

```js


const inorderTraversal = function (root) {
 
    const arr = []
    const deep = (root) => {
        if (!root) {
            return
        }
        
        deep(root.left)
        arr.push(root.val)
        deep(root.right)

    }
    deep(root)

    return arr


};
```

## 3. 后续遍历 (左->右->根)

所谓后续，就是根在最后，先处理左右，再处理根

```
D -> E -> B -> C -> A
```

```js


const postorderTraversal = function (root) {
 
    const arr = []
    const deep = (root) => {
        if (!root) {
            return
        }
        
        deep(root.left)
        deep(root.right)
        arr.push(root.val)

    }
    deep(root)

    return arr


};
```

# 广度优先遍历 （层序遍历）

广度优先遍历，就像是从上往下，按层次深度依次访问，同一层节点从左到右遍历，核心是 **先广后深**

其实可以理解为，按行扫描，先从第一层的根开始，再从第二层，第三层，依此类推

```
A -> B -> C -> D -> E
```

```js


const levelOrder = function (root) {
    if (!root) {
        return []
    }
    const arr = []
    const deep = (root, level) => {
        if (!root) {
            return
        }
        if (!arr[level]) {
            arr[level] = []
        }
        arr[level].push(root.val)
        deep(root.left, level + 1)
        deep(root.right, level + 1)
    }
    deep(root, 0)
    return arr
};
```
